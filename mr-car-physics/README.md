# MR Car Physics

Meta Quest 3で動作する、ハンドトラッキングとコントローラーで操作できるクラシックカーのMRアプリです。

## 機能

- 🚗 3Dクラシックカーモデルの表示
- 🤲 ハンドトラッキング対応
- 🎮 コントローラー操作対応
- 🌍 物理エンジン（Cannon.js）による重力シミュレーション
- 📱 QRコードでQuest 3から簡単アクセス

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. QRコードの生成

```bash
python3 generate_qr.py
```

これにより、`public/qr_code.png`にアクセス用のQRコードが生成されます。

### 3. 開発サーバーの起動

```bash
npm run dev
```

サーバーが起動したら、Quest 3のブラウザで表示されたQRコードをスキャンするか、
表示されたURLに直接アクセスしてください。

## 操作方法

### コントローラー操作

- **グリップボタン（握る）**: 車を掴んで移動
- **トリガーボタン**: 車を拡大（押している間）
- **グリップ解除**: 車を放す（物理法則に従って落下）

### ハンドトラッキング

- 右手の位置に車が出現します

### 初期配置

- 車は右コントローラーまたは右手の先端に出現します
- MRセッション開始後、自動的に配置されます

## 技術スタック

- **Three.js**: 3Dグラフィックス
- **Cannon-es**: 物理エンジン
- **WebXR**: VR/AR対応
- **Vite**: ビルドツール
- **vite-plugin-mkcert**: HTTPS対応（Quest 3で必須）

## 注意事項

- Quest 3のブラウザでアクセスする必要があります
- HTTPSが必須です（vite-plugin-mkcertで自動対応）
- 初回アクセス時にSSL証明書の警告が出る場合がありますが、「詳細設定」から進んでください

## トラブルシューティング

### 車が表示されない

1. ブラウザのコンソールでエラーを確認
2. 3Dモデル（`classic_car_model.glb`）が正しく配置されているか確認
3. WebXRのサポートを確認

### コントローラーが反応しない

1. Quest 3のコントローラーが正しくペアリングされているか確認
2. MRモードが正しく起動しているか確認

### 物理が動作しない

1. 車を掴んで離してみてください
2. ブラウザのコンソールでエラーを確認
